# 🎓 Student Dashboard

A Student Dashboard web application developed as part of my ICT Academy MERN Stack Full-Stack Development Internship. This project demonstrates full-stack web development using MongoDB, Express.js, React.js, and Node.js. The application allows users to manage student records through a simple and responsive interface.

## 🚀 Features

- View student details
- Add new student records
- Update existing student information
- Delete student records
- Responsive user interface
- REST API integration
- MongoDB database connectivity
- Real-time CRUD operations

## 🛠️ Tech Stack

### Frontend
- React.js
- JavaScript
- HTML5
- CSS3
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose
